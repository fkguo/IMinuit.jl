Example

fcn(x) = x[1]^2 + (x[2]-2)^2 + (x[3]-3.2)^4
m = Minuit(fcn, [1,0,1]; name=["a", "b", "c"], error=0.1*ones(3), fix_a=true, limit_b=(0, 50) )
migrad(m)
hesse(m)
Name Value Hesse Error Minos Error- Minos Error+ Limit- Limit+ Fixed
0 a 1.0 0.1 yes
1 b 2 1 0 50
2 c 3 4

Here the parameters are collected in an array x which is the argument of fcn(x), and typeof(m) = ArrayFit. In this case, one can use external code to compute the gradient as gradfun(par) = gradient(fcn, par) (the exported gradient function is from ForwardDiff), and include grad = gradfun as a keyword argument.

If fcn is defined as fcn(a, b), then the starting values need to be set as Minuit(fcn, a = 1, b = 0).
